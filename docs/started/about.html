# JNTemplate

关于JNTemplate

JNTemplate中文名称为极念模板引擎,JN是极念中文拼音首字母组成，极是极致，最佳的意思，念是理念，想法的意思。可以理解为最佳的想法，最棒的理念！

JNTemplate是一款完全国产的基于C#语言开发的跨文本解析引擎，它能生成任何基于文本的内容，你可以用它来解析web模板，也可以用来做代码生成器，XML文档生成器等任何你能想到的用途。

为什么选择JNTemplate

  1. 稳定：我们从.net framework 2.0 时代起，一直在坚持更新，进行功能改进与技术支持，稳定性也经过大家的验证。
  2. 快速：基于IL emit进行代码编译，不管是编译速度还是运行速度都表现优秀
  3. 免费：完全支持商用，不需要任何费用。
  4. 简单：简洁好用，学习成本低。

有了Razor为什么还需要JNTemplate

Razor做为微软官方视图引擎，在ASP.NET MVC中，它无疑是最好的选择。但是在某些场景，我们可能需要一款相对灵活，轻量的的模板引擎，所以他们之间是不冲突的。

# IConfig 接口

命名空间: JinianNet.JNTemplate.Configuration 程序集:JinianNet.JNTemplate.dll

Jntemplate引擎的配置参数

public interface IConfig

# 属性

属性 说明
ResourceDirectories 模板目录
Charset 字符编码，默认utf-8
TagPrefix 标签前缀，默认${
TagSuffix 标签后缀，默认}
TagFlag 简写标签标志，默认$
ThrowExceptions 运行时出错是否抛出异常(仅对解释型引擎生效)
StripWhiteSpace 是否处理标签前后空白字符
IgnoreCase 是否忽略大小写
EnableTemplateCache 是否启用模板缓存，建议启用(仅对解释型引擎生效)
DisableeLogogram 是否禁用标签 简写
Loader 资源加载器
TagParsers 标签解析器

# 说明

ResourceDirectories

当使用Engine.LoadTemplate(...)创建模板或者使用$load(...)与inclub(...)标签时，如果填写的是相对对路径，则在在配置的资源目录ResourceDirectories搜寻模板文件,如果未配置该项，则以上用法使用相对路径时会出现资源未找到的错误提示。

Charset

字符编码，默认使用utf-8,推荐使用utf-8

TagPrefix

标签前缀，默认为${,可以根据自己的喜好配置。

TagSuffix

标签前缀，默认为},可以根据自己的喜好配置。

TagFlag

简写标签标志，默认$，只能为一个char字符，如果不使用简写，可以通过DisableeLogogram项关闭简写模式。
